## Controls for the prototype levels 
F1 – First Map (Maple Leaves)
F2 – Second Map (Snow Mountains)
C – Cinematic Mode/First Person Mode Toggle
Arrow Keys Left /Right – Transition between set camera’s / views
In first person mode:
Right click to zoom in
Shift to run faster
